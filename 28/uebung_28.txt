Übung 28
========

Verwenden Sie das zip-Archiv : git_28.zip

a)
Gehen Sie in das Verzeichnis "git/client01/brep01".

Erstellen Sie einen Branch "client01" und führen an der bundesliga.txt-Datei noch eine Änderung durch.
Committen Sie Ihre Änderungen und übertragen alles (incl  Branch) ins zentrale Repository.



b)
Gehen Sie in das Verzeichnis "git/client02/brep01".
Erstellen Sie einen Branch "client02" und übertragen den Branch ins zentrale Repo.
Legen Sie eine neue Datei aktivitaeten.txt an (darf leer sein).
Committen Sie Ihre Änderungen und übertragen alles ins zentrale Repository.


c)
rebasen Sie den branch client02 in den master.
 
d)
Gehen Sie in das Verzeichnis "git/client03/brep01".
Erstellen Sie einen Branch "client03" und führen an der bundesliga.txt-Datei noch eine Änderung durch;
aber anders als in a).


Committen Sie Ihre Änderungen und übertragen alles (incl  Branch) ins zentrale Repository.
Rebasen Sie anschließend den Branch in den master.

e)
rebasen Sie den Branch client01 in den Master.



Tipps:
git branch
git rebase
git remote
git add
git commit 

touch bundesliga.txt   (erzeugt eine neue leere Datei)
ll (Ansicht der im aktuellen Verzeichnis befindlichen Dateien)
