Übung 04
========

Verwenden Sie das zip-Archiv : git_04.zip

a)
Clonen Sie das Repository "git/repos/brepo01" in "git/client01",
sowie in "git/client02".

In den Verzeichnissen "git/client01/brepo01" sowie "git/client02/brepo02" befindet sich
eine leere Datei "Bundesliga.txt".

b)
Öffnen Sie die Bundesliga.txt-Datei aus "git/client01/brepo01" und fügen eine beliebige Zeile
hinzu.

Sichern Sie die Änderung im lokalen Repository.
(vergessen sie NICHT die commit-Message (!))

c)
Legen sie in "gilt/client02/brepo01" eine neue Datei an, "Lieblingsessen.txt".
Die Datei kann leer sein; wenn Sie wollen, können Sie aber auch ein paar Zeilen Text hinzufügen.

Sichern Sie die Änderung im lokalen Repository.
(vergessen Sie nicht die commit-Message (!))

e)
Clonen Sie nun das Repository "git/repos/brepo01" in "git/client03".
Schauen Sie sich das Arbeitsverzeichnis "git/client03/brepo01" an.


Tipps:
git remote
git status
git clone
git add
git commit 
git config

touch bundesliga.txt   (erzeugt eine neue leere Datei)
